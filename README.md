- Documentation: see `docs/PUBLIC_APIS_COMPONENTS.md` for full API/component docs, examples, and usage instructions.
- Quick start (local preview):

  ```bash
  cd /workspace
  python3 -m http.server 8080
  # open http://localhost:8080/
  ```
